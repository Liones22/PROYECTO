# PROYECTO

Dashboard financiero personal en HTML, CSS y JavaScript.

## Funcionalidades

- Registro de ingresos y egresos con categorías dinámicas.
- Resumen con ingresos, egresos, balance y tasa de ahorro.
- Filtros por texto y tipo de transacción.
- Eliminación de movimientos individuales.
- Presupuesto mensual con alertas de uso.
- Persistencia local con `localStorage`.
- Carga de datos de ejemplo y limpieza completa.
- Exportación de transacciones a CSV.
- Visualización con gráficos de conceptos y comportamiento mensual.

## Uso

1. Abre `index.html` en tu navegador.
2. Agrega transacciones desde el formulario.
3. Usa filtros, presupuesto y exportación para gestionar tu historial.
